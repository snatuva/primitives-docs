Headless Utility: The @snatuva/primitives library is purely a headless utility. It does not manage any CSS. Instead, it wires up the correct ARIA attributes, handles keyboard navigation, and provides data-* attributes for you to hook into. You have complete control over the visual presentation using your preferred CSS solution.
